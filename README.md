# VK Quiz

VK Quiz — веб-приложение для создания и проведения интерактивных квизов в реальном времени.

Проект позволяет организаторам создавать квизы, запускать live-комнаты, показывать вопросы участникам синхронно и получать итоговый лидерборд. Участники могут проходить квизы по коду комнаты, участвовать в live-режиме или проходить опубликованные квизы самостоятельно.

## Основные возможности

- регистрация и авторизация пользователей;
- выбор роли: участник или организатор;
- создание и редактирование квизов;
- поддержка текстовых вопросов и вопросов с изображениями;
- одиночный и множественный выбор ответа;
- публичные и приватные квизы;
- подключение участников к live-квизу по коду комнаты;
- QR-код для быстрого входа в комнату;
- синхронный показ вопросов через WebSocket;
- подсчёт баллов и финальный лидерборд;
- solo-режим для самостоятельного прохождения;
- рейтинг квизов;
- возможность изменить свою оценку;
- личный кабинет участника и организатора;
- история квизов и результатов;
- аналитика для организатора;
- projector mode для ведущего;
- PWA-режим для установки сайта на телефон;
- адаптация под мобильные устройства.
